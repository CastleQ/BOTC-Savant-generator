# 🎩 시계피 서번트 명제 생성기 (BOTC Savant Generator)

보드게임 **'Blood on the Clocktower (시계탑에 흐른 피)'**의 캐릭터 중 하나인 **서번트(Savant)**를 위한 명제 생성 도구입니다.
이야기꾼(Storyteller)이 게임 진행 중 실시간으로 적절한 명제를 떠올려야 하는 부담을 줄이기 위해 제작되었습니다.

## 🛠 사용 방법
1. 웹사이트에 접속합니다.
2. `[명제 생성]` 버튼 또는 서번트 아이콘을 클릭합니다.
3. 화면 상단에 생성된 명제를 확인하고 플레이어에게 제공합니다.
4. 필요시 하단 드롭다운 메뉴에서 언어를 변경할 수 있습니다.
