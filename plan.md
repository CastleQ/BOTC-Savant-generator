# 프로젝트: 시계피 서번트 명제 생성기 (BOTC Savant Generator)

이 문서는 개발 과정을 추적하는 할 일 목록입니다. TDD(테스트 주도 개발) 방식에 따라 하나씩 체크하며 진행합니다.

## 1. 프로젝트 설정 (Project Setup)
- [ ] 기본 파일 생성 (index.html, style.css, script.js)
- [ ] 테스트 환경 설정 (간단한 콘솔 테스트 준비)

## 2. 데이터 준비 (Data Preparation)
- [ ] 제공된 CSV 데이터를 자바스크립트 배열로 변환하여 저장 (data.js)
- [ ] 데이터가 올바르게 불러와지는지 테스트

## 3. 핵심 로직 구현 (Core Logic - TDD)
- [ ] Test 1: 데이터베이스에서 무작위 명제 하나를 가져올 수 있다.
- [ ] Test 2: 한글/영어 언어 설정에 따라 올바른 데이터를 가져온다.
- [ ] Test 3: (고급) 직전에 나온 명제 3개는 다시 나오지 않는다 (중복 방지).

## 4. UI 구조 잡기 (HTML Structure)
- [ ] 기본 레이아웃 작성 (아이콘, 텍스트 영역, 버튼)
- [ ] 외부 링크 및 푸터 영역 작성

## 5. 스타일링 (Styling & CSS)
- [ ] 중앙 정렬 및 반응형 레이아웃 구성
- [ ] 서번트 아이콘 회전 애니메이션 구현
- [ ] 다크모드/라이트모드 자동 감지 적용

## 6. 기능 통합 (Integration)
- [ ] 버튼 클릭 시 명제 생성 로직 연결
- [ ] 언어 변경 드롭다운 기능 연결
- [ ] 명제 생성 시 애니메이션 효과 (선택사항)

## 7. 배포 (Deployment)
- [ ] 최종 코드 점검 및 정리 (Tidy First)
- [ ] Github Pages로 배포
